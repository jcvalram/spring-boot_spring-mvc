Te explicaré la funcionalidad del módulo de Solicitudes:

En la vista detalle.html vas a mostrar un texto que diga por ejemplo "Regístrese para Aplicar para esta Vacante". 
Este texto solo deberás mostrarlo a los usuarios anónimos.

En la vista detalle.html después de mostrar los detalles de la Vacante hay un botón HTML llamado Aplicar. 
Este botón HTML solo deberás mostrarlo para usuarios autenticados. El punto clave de este botón es formar una URL 
dinámica para que se envíe el idVacante al controlador SolicitudesController.

El diseño del formulario HTML que utilizarás para que un usuario envíe la solicitud será el archivo formSolicitud.html 
que esta en tu carpeta de la plantilla.
